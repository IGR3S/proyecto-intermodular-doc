# Introducción

Este proyecto corresponde a la página web oficial de una pastelería local, diseñada para mostrar sus productos, servicios y facilitar el contacto con los clientes de forma sencilla y visual.

## Objetivo del proyecto

El objetivo del proyecto es proporcionar una presencia digital a la pastelería, permitiendo a los clientes conocer los productos disponibles, horarios, ubicación y formas de contacto.

## Problema que resuelve

La página web resuelve la falta de visibilidad online del negocio, centralizando la información principal y evitando que los clientes dependan únicamente de redes sociales o visitas presenciales.
