## Dificultades

Una de las principales dificultades fue diseñar una interfaz atractiva que represente la identidad de la pastelería y sea adaptable a distintos dispositivos.

## Aprendizajes

Durante el desarrollo del proyecto se aprendió a:

- Crear sitios web estáticos orientados a negocios locales
- Documentar proyectos usando Markdown
- Publicar páginas web mediante GitHub Pages

## Posibles mejoras

- Añadir un formulario de pedidos online
- Integrar un mapa interactivo
- Incorporar una galería de imágenes más completa
- Conectar la web con redes sociales
